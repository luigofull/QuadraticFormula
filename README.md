# My Manim animation about quadratic formula.

## How to compilate animation:
> 1. clone the repo
> 2. in project, create python venv
>
> `python -m venv .venv`
>
> 3. in venv install manim lib
>
> `pip install manim`
>
> then go to /project-dir/src/sketch
> and stasrt the compitation
>
> `manim --quality=k "sketch.py"`
